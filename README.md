# 我的Android期中作業 - MyCafe
> 時間：2022/11
---
### 內容：
> 這個APP是我在大三上所做的期中作業，此時是我學習Android Studio 將近兩個月的時間，
此App主要是要設計給一般使用者所使用的咖啡廳應用程式，此App可讓使用者預先儲值現金，
然後進行訂餐、還可以得知咖啡廳的最新消息、網路查詢功能、購物車功能、購買紀錄、會員資料…等等。
---
### 包含元件：
>#### Fragment
>#### Dialog
>#### Navigation Drawer
>#### collapsing Toolbar
>#### Spinner
>#### FAB

>#### Scrolling
